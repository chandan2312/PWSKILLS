1. `p`: Uses for a paragraph.
  2. `h1`, `h2`, `h3`, ... `h6`: Define level of headings.
  3. `a`: hyperlinks.
  4. `img`: To display images through link or path.
  5. `ul`: unordered list with various types of dot notations.
  6. `ol`: ordered list with numerical, alphabetical or Roman count .
  7. `li`: Represent single item from a list group.
  8. `div`: Defines a division or a container for other elements.
  9. `span`: Defines an inline container for a small piece of content or text.
  10. `table`, `tr`, `td`: Used for creating tables and table rows and cells.
