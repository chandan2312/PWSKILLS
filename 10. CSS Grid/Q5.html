CSS Grid: justify-items and justify-self

justify-items:

Applied to the grid container.

Sets the default horizontal alignment for all items within the grid.

This property aligns all the items in the grid to a common horizontal position.

justify-self:

Applied to individual grid items.

Sets the horizontal alignment for a specific grid item within the grid.

Allows you to override the default alignment set by justify-items for a specific item.

In the example, the justify-items property is used to center all grid items horizontally within the grid container. However, the justify-self property is applied to a specific item, and it overrides the horizontal alignment for that item, setting it to "start."

As a result, all items in the grid will be horizontally centered due to justify-items, except for the item with the class "subcontainer" which will align to the start of its grid cell.

You can use these properties to control the horizontal alignment of items within your grid, with justify-items setting a default for all items and justify-self allowing you to apply individual alignment to specific items.
