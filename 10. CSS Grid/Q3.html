grid-auto-row and grid-auto-column are properties in CSS Grid that allows to define the size of implicitly created rows and columns in a grid container. Implicit rows and columns are those that are not explicitly defined using the grid-template-rows and grid-template-columns properties.

CSS Grid: grid-auto-row and grid-auto-column

grid-auto-row:

Defines the size of implicitly created rows in a grid.

Applied when the number of rows created exceeds those explicitly defined using grid-template-rows.

Controls the height of these additional, implicitly generated rows.

grid-auto-column:

Defines the size of implicitly created columns in a grid.

Applied when the number of columns created exceeds those explicitly defined using grid-template-columns.

Controls the width of these additional, implicitly generated columns.

For example, if you have a grid container with explicit rows and columns defined and add more items than the specified rows or columns, the grid will automatically generate additional rows or columns. The size of these additional rows or columns is determined by grid-auto-row and grid-auto-column.

You can use these properties to customize the dimensions of implicitly generated rows and columns, allowing for flexible grid layouts.
